# Coaching Platform - Setup & Account Creation

## Prerequisites - Create These Accounts

Follow these steps to set up all necessary accounts:

### 1. **GitHub Account** (if you don't have one)
- Link: https://github.com/signup
- Purpose: Store code repository
- Time: 2 minutes

### 2. **Supabase Account** (Backend + Database)
- Link: https://app.supabase.com
- Sign up with GitHub (recommended)
- Purpose: PostgreSQL database + Authentication + Real-time
- Free tier includes:
  - 500 MB storage
  - 2 GB egress/month
  - Unlimited API calls
- Time: 5 minutes

### 3. **Vercel Account** (Frontend Hosting)
- Link: https://vercel.com/signup
- Sign up with GitHub (recommended)
- Purpose: Deploy Next.js frontend
- Free tier includes:
  - Unlimited deployments
  - 100 GB bandwidth/month
  - Automatic HTTPS & SSL
- Time: 2 minutes
